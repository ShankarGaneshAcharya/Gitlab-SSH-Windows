# Gitlab-SSH-Windows

[SSH Key Setup for GitLab on Windows: A Developer’s Guide](https://github.com/ShankarGaneshAcharya/Gitlab-SSH-Windows/blob/main/SSH%20Key.md)

This guide explains how to connect GitLab with Visual Studio Code using the GitLab Workflow extension and a Personal Access Token (PAT). You'll learn how to install the extension, generate a secure token, authenticate your GitLab account, and access GitLab features like issues and merge requests directly within VS Code.

[Configure GitLab Personal Access Token in Visual Studio Code](https://github.com/ShankarGaneshAcharya/Gitlab-SSH-Windows/blob/main/PAT.md)

This guide explains how to securely set up SSH key authentication for GitLab on a Windows machine using Git Bash. You'll learn how to generate a new SSH key pair, add it to the SSH agent, configure your SSH settings, and connect to GitLab — all with clear, step-by-step instructions tailored for developers.
